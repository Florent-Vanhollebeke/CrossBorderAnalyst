## 📋 Procédure de mise à jour recommandée

1. **Novembre 2025** ✅ — nouveau PASS France publié → mis à jour dans tous les fichiers RAG
2. **Décembre 2025** ✅ — nouveau SMIC France → mis à jour dans tous les fichiers RAG
3. **Janvier 2026** ⏳ — taux AVS/LPP/AC Suisse publiés par OFAS → vérifier fiches cantonales
4. **Février 2026** ⏳ — vérifier taux IS effectifs 2026 sur ESTV (estv.admin.ch)
5. **Courant 2026** ⚠️ — suivre révision franchise TVA France (prochaine révision triennale)
6. **Septembre 2025** ⚠️ — résultat vote Bâle-Ville Handänderungssteuer (1,5%?) à intégrer dans `feuille_cantonale_bs.txt` et `droit_mutation_ch.txt`
7. **Après chaque mise à jour significative** — reconstruire l'index FAISS :
   ```bash
   cd backend/ml_training
   python index_faiss.py
   ```
8. **Redéployer** le container backend :
   ```bash
   docker-compose up -d --build backend
   ```
